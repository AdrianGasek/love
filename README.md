## O projekcie słów kilka:

Projekt realizowany jest co roku, przed albo w trakcie konkretnego dnia a mianowicie w Walentynek (14 luty).
Co roku robię stronę typu kartkę walentynkową dla swojej kochanej kobiety, zawierającą słowa prosto z serca.
Bedę starał się robić jak najładniejsze strony każdego roku za równo jeśli chodzi o tworzenie grafik jak i sam kod.
Do każdej ze stron bedę używał różnych bibliotek Reacta tak aby podnosić sobię poprzeczkę w kodowaniu i twórzyć unikalne wyglądy każdej z walentynek. Start projektu Luty 2024. Każda z walentynek będzie numerowana w adresie strony rocznikowo (np. adres_strony/2024)
